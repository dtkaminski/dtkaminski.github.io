feat(dashboard): add Sign out to the greta app bar

The only logout lived in the /app shell header, which is hidden once the rich
dashboard iframe takes over — so a signed-in user had no visible way to sign
out. Add a "Sign out" button to the dashboard app bar (far right), shown only
in an authenticated workspace (OI_ASK present; hidden in the public demo where
there's no session to end).

On click: end the Supabase session (window.FRKL_LIVE.sb.auth.signOut(), with a
belt-and-braces localStorage clear of the sb-*-auth-token + demo passcode gate
so logout holds even if the client instance isn't reachable), then send the top
window (this dashboard runs in a same-origin iframe) to the homepage (/).
