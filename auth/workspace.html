Connect Klaviyo

Paste a read-only Private API Key — takes about 30 seconds:

  1. Open Klaviyo → Settings → API keys ↗
  2. Click Create Private API Key, name it “Operator Intelligence”
  3. Choose Read-only access (or scopes: accounts, metrics, flows, campaigns, profiles)
  4. Copy the key (starts with pk_) and paste it below

🔒 We validate it, then only ever read — we never send, edit lists, or change anything in your Klaviyo.
